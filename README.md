# Malawi Urban Walkability Analysis

A database-driven web application analyzing walkability in Malawian cities.
#the data folder is ignored
## Datasets
- Malawi Roads Network
- Malawi District Boundaries  
- Malawi Schools
- Malawi Health Facilities

## Setup
1. Install requirements: `pip install -r requirements.txt`
2. Set up PostgreSQL with PostGIS
3. Run the analysis scripts
